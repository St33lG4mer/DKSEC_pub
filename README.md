# DKSec

DKSec is a threat detection and rule engineering toolkit for building, translating, scoring, and validating detection content.

## Quick Start

1. Create and activate a virtual environment.
2. Install dependencies.
3. Run the app or scripts you need.

```powershell
pip install -r requirements.txt
```

## Security

Security guidance and incident handling are documented in [SECURITY.md](SECURITY.md).

Before committing changes, run:

```powershell
pre-commit run --all-files
```

CI also performs repository secret scanning with Gitleaks.
